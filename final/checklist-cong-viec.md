# Checklist Công Việc Đồ Án Cuối Kỳ - DS317 Khai Phá Dữ Liệu

## Tổng Quan Về Đồ Án

- [ ] Giới thiệu đề tài, định nghĩa bài toán (input và output)
- [ ] Mô tả ứng dụng thực tế của đề tài
- [ ] Phân tích khó khăn và thách thức
- [ ] Xác định mục tiêu và phạm vi thực hiện

## Nghiên Cứu Liên Quan

- [ ] Khảo sát các công trình nghiên cứu tương tự
- [ ] Phân tích kết quả khảo sát
- [ ] Xác định định hướng triển khai đề tài

## Cơ Sở Lý Thuyết

- [ ] Trình bày nền tảng lý thuyết về dữ liệu lớn
- [ ] Giới thiệu các công nghệ sẽ áp dụng
- [ ] Mô tả kiến trúc dữ liệu lớn trong doanh nghiệp

## Phân Tích Bộ Dữ Liệu

- [ ] Tìm hiểu và khám phá dữ liệu (EDA)
- [ ] Chuẩn bị và tiền xử lý dữ liệu
- [ ] Phân tích vấn đề và xác định yêu cầu

## Phương Pháp Đề Xuất

- [ ] Thiết kế framework tổng quan từ input đến output
- [ ] Thiết kế kiến trúc dữ liệu lớn
- [ ] Phát triển hệ thống thông minh ứng dụng AI (nếu có)

## Thực Nghiệm

- [ ] Chuẩn bị bộ dữ liệu thực nghiệm
- [ ] Tổ chức dữ liệu trên kiến trúc dữ liệu lớn
- [ ] Thiết kế độ đo đánh giá
- [ ] Thực hiện kịch bản thực nghiệm theo thời gian
- [ ] Đánh giá và phân tích kết quả thực nghiệm

## Yêu Cầu Cơ Bản (Bắt Buộc)

- [ ] Xây dựng ứng dụng BI cho bài toán đề tài
- [ ] Giám sát dữ liệu input
- [ ] Đo lường chất lượng dữ liệu
- [ ] Áp dụng phương pháp Machine Learning từ tài liệu tham khảo

## Yêu Cầu Nâng Cao (Tùy Chọn)

- [ ] Áp dụng phương pháp cải thiện chất lượng dữ liệu:
  - [ ] Điền khuyết dữ liệu
  - [ ] Tăng cường dữ liệu
  - [ ] Làm giàu dữ liệu
- [ ] Nghiên cứu cải tiến phương pháp ML/DL
- [ ] Triển khai chức năng demo cho output bài toán
- [ ] Thực nghiệm trên kiến trúc dữ liệu lớn

## Sản Phẩm Cuối Cùng

- [ ] Báo cáo đề tài (docx và pptx)
- [ ] Bộ dữ liệu thực nghiệm
- [ ] Toàn bộ source code
- [ ] Video thuyết trình (tất cả thành viên tham gia, mở camera)

## Kết Luận và Hướng Phát Triển

- [ ] Đánh giá ưu nhược điểm của phương pháp đề xuất
- [ ] Xác định định hướng phát triển tiềm năng

## Quản Lý Dự Án

- [ ] Phân công nhiệm vụ cho từng thành viên
- [ ] Lập kế hoạch thời gian thực hiện
- [ ] Giám sát tiến độ và điều chỉnh khi cần thiết
- [ ] Đảm bảo chất lượng sản phẩm cuối cùng
